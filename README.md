# plop-templates

Ready Plop configurations

## Install plop

```bash
npm install --save-dev plop
```

or

```bash
yarn add -D plop
```

or

```bash
pnpm add -D plop
```

---

## Configuration

1. Add this to package,json

```json
{
    ...,
    "scripts": {
        "plop": "plop"
    },
    ...
}
```

2. copy the contents of one of these folders to the root of your project

---

## Run

```bash
npm run plop
```

or

```bash
yarn plop
```

or

```bash
pnpm run plop
```
